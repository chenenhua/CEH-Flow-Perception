# Code of Conduct
# 行为准则

## 1. Our Goal
## 1. 我们的目标

We want this project to be a place for serious engineering discussion, open technical exploration, and respectful collaboration.

我们希望这个项目成为一个适合严肃工程讨论、开放技术探索和相互尊重协作的地方。

## 2. Expected Behavior
## 2. 期望行为

Please be respectful, specific, and evidence-based.

请保持尊重、具体和基于证据的沟通方式。

Please focus on technical issues rather than personal attacks.

请聚焦技术问题，而不是针对个人进行攻击。

Please assume good faith where reasonable.

在合理范围内，请尽量假定对方是善意的。

## 3. Unacceptable Behavior
## 3. 不可接受行为

The following behavior is not acceptable:

以下行为不可接受：

- harassment or abuse
- 骚扰或辱骂

- personal attacks
- 人身攻击

- discrimination
- 歧视性言论

- malicious disruption of discussion
- 恶意破坏讨论秩序

- knowingly false accusations
- 明知不实的指控

- repeated off-topic provocation
- 持续的无关挑衅

## 4. Technical Disagreement
## 4. 技术分歧

This project welcomes disagreement about architecture, algorithms, legal strategy, performance claims, and engineering direction.

本项目欢迎围绕架构、算法、法律策略、性能结论和工程方向展开分歧讨论。

However, disagreement should be supported by logs, code, benchmarks, papers, reproducible tests, or reasoned analysis.

但分歧应尽量由日志、代码、基准测试、论文、可复现实验或有理有据的分析支撑。

## 5. Enforcement
## 5. 执行方式

Project maintainers may remove content, lock threads, reject contributions, or block users whose conduct materially harms the project environment.

如果某些行为实质性破坏项目环境，项目维护者可以删除内容、锁定讨论、拒绝贡献或屏蔽相关用户。
